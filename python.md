# Переменные уровня модуля

1C:
```bsl
Перем Количество;
Перем Сумма;

Процедура Тест()
    Цена = 100;
    Сумма = Цена * Количество;
КонецПроцедуры

Количество = 5;
Сумма = 0;
```

```python
quantity = 5
amount = 0

def test():
    price = 100
    global amount # это указание что мы хотим изменить глобальную переменную
    amount = price * quantity
```
